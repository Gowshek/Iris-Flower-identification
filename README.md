# 🌸 Iris Flower Identification

## Description
A Machine Learning project that predicts the species 
of an Iris flower based on its measurements using 
the KNN algorithm.

## Libraries Used
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## ML Algorithm
- K Nearest Neighbors (KNN)
- n_neighbors = 3

## Dataset
- Built-in Iris dataset from Scikit-learn
- 150 flowers, 3 species, 4 measurements

## What the Model Does
- Trains on 120 flowers
- Tests on 30 flowers
- Achieves 100% accuracy!
- Predicts species from user input

## Species Predicted
- 🌺 Iris Setosa
- 🌸 Iris Versicolor
- 💜 Iris Virginica

## Key Learnings
- ML Pipeline
- Train Test Split
- KNN Algorithm
- Model Training & Testing
- User Input Prediction

## How to Run
1. Open the notebook in Google Colab
2. Run all cells from top to bottom
3. Enter flower measurements when prompted
4. Get your prediction! 🌸
